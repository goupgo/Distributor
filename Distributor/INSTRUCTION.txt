#A SIMPLE FILE DISTRIBUTOR APP 
VERSION 0.1
BEFORE USE THE .EXE APPLICATION, PLEASE MOVE TWO FOLDERS "Mess" and "Organised' TO DESKTOP.
INSTRUCTION:
+ MOVE ALL FILE WITH .PNG AND .MP4 INTO "Mess" FOLDER, THEN RUN THE .EXE FILE
  --> ALL FILES SHOULD BE ALLOCATED IN "Organised" IN THE RIGHT PLACE.

P/S: THIS VERSION 0.1 ONLY HAS PNG AND MP4, THE APP WILL BE UPDATED LATER.
P/S: MOVE THE DISTRIBUTOR FOLDER ON DESKTOP, THE APPLICATION CAN ONLY RUN IN THE DISTRIBUTOR FOLDER., DON'T MOVE IT ELSE WHERERE OR ELSE IT WON'T WORK.
